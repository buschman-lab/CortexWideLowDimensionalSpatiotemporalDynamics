# Cortex Wide Spatiotemporal Dynamics 
This reposity contains example data and code to generate figures from the following paper

Camden J. MacDowell, Timothy J. Buschman,
Low-Dimensional Spatiotemporal Dynamics Underlie Cortex-wide Neural Activity,
Current Biology, 2020, ISSN 0960-9822,https://doi.org/10.1016/j.cub.2020.04.090.
Contact: camdenm@princeton.edu for questions

## Use
To generate figures download and unzip the reposity and navigate to the repository directory. Then run the GenerateFigures() function in MATLAB 2018b of later. See GenerateFigures() function for details on figures.
